Política de Privacidade

Última atualização: 25 de Fevereiro de 2026

1. Coleta de Dados

O plugin **Shorts API** não coleta dados pessoais dos visitantes do seu site WordPress. Ele processa apenas metadados técnicos (título, URL, categorias e anexos de vídeo) para fins de sincronização com o backend configurado pelo administrador.

2. Uso da API

Os dados sincronizados são enviados exclusivamente para o endpoint da API configurado no painel administrativo do plugin. Recomendamos o uso de conexões seguras (HTTPS) para todos os envios.

3. Cookies

Este plugin não armazena cookies no navegador dos visitantes. Qualquer cookie gerado no frontend (Next.js/React) é de responsabilidade da aplicação que consome a API.

4. Segurança

Implementamos autenticação via API Key para garantir que apenas sites autorizados possam enviar dados para o seu backend Express.
