Registers a public REST endpoint to list posts with video attachments using a custom database table for maximum performance. Designed to power TikTok-style shorts frontends.

== Description ==

Shorts API Optimized is a high-performance plugin designed for creators who want to build TikTok-style video feeds using WordPress as a backend. It automatically synchronizes posts containing videos into a custom database table, allowing for lightning-fast queries that bypass the overhead of standard WP_Query.

= Features =
* Optimized REST API endpoint (`/wp-json/shorts/v1/posts`).
* Automatic synchronization on post save/update.
* WP-CLI commands for bulk synchronization.
* Custom settings page for logo, favicon, and SEO titles.
* Dynamic domain verification system for multi-site setups.

== Installation ==

1. Upload the `shorts-api` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings under 'Shorts API' in the admin menu.
